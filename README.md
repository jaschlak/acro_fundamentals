# Drills

## Repetition Drills
- repeat same entry 10 times
- repeat same exit 10 times
- no talking flow practice
- slow tempo balance drills
- one new transition per jam

## Balance Drills
- stable bird hold
- stable side star hold
- one-arm balance assistance (bird,sidestar,star)
- mono variations
- smooth flyer mounting

---

# Communication

- "ready?"
- "up"
- "down"
- "spot"
- communicate fatigue honestly
- slow before fast

---

# Jam Goals

- focus on flow over tricks
- smooth entries and exits
- low reset time
- build repeatable sequences
- learn one new thing each jam (after fundamentals comfortable)
